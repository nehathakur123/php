Margin

The margin is an outer, invisible border around your element.

The default value for the margin properties is auto, which usually translates to zero

The header elements (h1-h6), the body tag and paragraphs - all of these usually have margins added to them by the internal stylesheet of the browser

Four types of margins : margin-top, margin-right, margin-bottom and margin-left
