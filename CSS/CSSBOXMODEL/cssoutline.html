Outline

With the outline properties, one can get an extra border, for extra visual attention for the element

Hello, world!

Differences between border and outline

  • We cannot apply a different outline width, style and color for the four sides of an element, like we can with the border - the values provided will be used for all four sides of the element

  • The outline is not a part of the element's dimensions, like the border is. This also means that the Browser won't reserve the required space for the outline

Shorthand properties of outline are : outline-width, outline-style and outline-color
