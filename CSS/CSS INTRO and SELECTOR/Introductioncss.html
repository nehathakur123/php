CSS

CSS is short for Cascading Style Sheets and is the primary language used for formatting webpages across the Internet and documents of markup (e.g. HTML and XML) in general. HTML is used for Document Content and CSS for Document Presentation.This allows the developer to re-use formatting rules across several places in the same document and even across multiple documents.

CSS types

  • Inline CSS
  • Internal CSS
  • External CSS
