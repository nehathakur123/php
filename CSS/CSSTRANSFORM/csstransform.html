Transforms in CSS3

The CSS3 transitions modules introduces four new properties

  1. CSS3 transition-property

    The transition property specifics which property is to be transformed - color, background, border etc.

  2. CSS3 transition-duration

    This property specifies how many second or milliseconds a transition takes to complete. The default is 0 and a good advice is to never specify a duration shorter than 5-6 milliseconds.

  3. Css3 transition-timing-function

    • Linear
    • Ease-in - Specifies a transition with a slow start
    • Ease-out - Specifies a transition with a slow end
    • Ease-in-out - Specifies a transition with a slow start and end.
    • Cubic-beizer(n,n,n,n) - Defines your own transition timing. Possible values ranges from 0 to 1.

Example1

<style> .bg-transition { padding: 15px; text-align: center; background-color: Crimson; width: 90%; min-height: 25px; transition: all 2s ease-in 0; -webkit-transition: all 2s ease-in 0; } .bg-transition:hover { background-color: CornFlowerBlue; } </style> <p class="bg-transition"> </p>
