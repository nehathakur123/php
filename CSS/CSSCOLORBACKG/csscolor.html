Colors and Background colors in CSS3

  • Text and Background colors

    • <p style="color: Red;">Red text</p> <p style="color: Blue;">Blue text</p> <p style="background-color: Silver; color: Red;">Red text</p> <p style="background-color: Gold; color: Blue;">Blue text</p>

  • Colors in CSS3


    • <p style="color: rgb(0,0,0);">Black text</p> <p style="color: rgb(255,255,255); background-color: rgb(0,0,0);">White text</p> .box { background-color: rgba(128,128,128,0.3); } <p style="color: #000000;">Black text</p> <p style="color: #0000ff;">Blue text</p> <p style="color: #ff0000;">Red text</p> <p style="color: red;">Red text</p>

  • Background images

    • .box { background-image: url('http://css3-tutorial.net/images/blu_stripes.png');}

  • The background-repeat property

    • <style type="text/css"> .box { background-image: url('http://css3-tutorial.net/images/blu_stripes.png'); } </style> <div class="box" style="background-repeat: repeat;"> Repeat </div> <div class="box" style="background-repeat: repeat-x;"> Repeat-X </div> <div class="box" style="background-repeat: repeat-y;"> Repeat-Y </div> <div class="box" style="background-repeat: no-repeat;"> No-Repeat </div>

  • The background-position property

    • <style type="text/css"> .box { background-image: url('http://css3-tutorial.net/images/blu_stripes.png'); background-repeat: no-repeat; background-position: top center; } </style>
