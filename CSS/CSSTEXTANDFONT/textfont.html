Text formatting in CSS3

  • font-family:

    • <p style="font-family:serif;">This is your browsers default serif font</p>

    CSSfontstack.com gives list of different font families

    body { font-family: Times New Roman, Times, serif; }

  • font-weight


    • The font-weight property defines how bold text are is and there are a lot of possible values; normal, bold, bolder, lighter, 100, 200, 300, 400, 500, 600, 700, 800, 900.
    1. 100: Thin, Hairline, Ultra-light, Extra-light
    2. 200: Light
    3. 300: Book
    4. 400: Regular, Normal, Plain, Roman, Standard
    5. 500: Medium
    6. 600: Semi-bold, Demi-bold
    7. 700: Bold
    8. 800: Heavy, Black, Extra-bold
    9. 900: Ultra-black, Extra-black, Ultra-bold, Heavy-black, Fat, Poster

    <p style="font-weight: 100;">Paragraph1 </p>

  • font-style

    • p.styleItalic {font-style: italic } p.styleOblique {font-style: oblique }

  • font-size

    • <style> .textLarge {font-size: 2em;} .textRegular{font-size:1em;} .textSmall{font-size:0.7em;} </style> <p class="textLarge"> Paragraph1 </p> <p class="textRegular"> Paragraph2</p> <p class="textSmall"> Paragraph3 </p>

  • word and letter spacing, align property

    • <style> p {text-align: right;} </style> <style> h3 {text-align: center;} </style> <style> p {text-align: justify;} </style> <style> h2 {word-spacing: 1.2em;} .tooMuch { word-spacing: 1.2em;} </style> <style> h2 {letter-spacing: 0.25em;} .tooMuch { letter-spacing: 0.5em;} </style>

  • text-shadow

    1. color
    2. x-coordinate
    3. y-coordinate
    4. blur radius

    • <style> .simpleShadow { color: #000; font-size: 2em; text-shadow: 1px 1px 5px #fff; } </style> <p class="simpleShadow">Simple shadow effect</p>

  • text-decoration

    • <div style="text-decoration: underline;">Hello, world!</div> <div style="text-decoration: line-through;">Hello, world!</div> <div style="text-decoration: overline underline;">Hello, world!</div>

  • text-indent

    • <style type="text/css"> p { text-indent: 50px; } </style>
